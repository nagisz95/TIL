# randomness
- Math.random()
    - ０〜１の間でランダムな数字を提供。
    - 例えば、０〜１０までのランダムな数字を得たい時、Math.random() * 10のように使用できる。
    - 返還される値は小数点まで表示されるため、float->integerしてくれる関数を一緒に使用する。


      - Math.round(number) -> 「1.2」を「1」に、「1.6」を「2」に四捨五入。
      - Math.ceil(number) -> 「1.2」を「２」に切り上げる。
      - MAth.floor(number) -> 「1.6」を「1」に切り下げる。
